#### 配置

Tag页面的字体大小：

```css
#full-tags-list .tag-entry a {
  font-size: 18px;
}
```



更改尺寸大于768的显示标题大小：

```css
@media only screen and (min-width: 768px) {
  .post-preview .post-title {
    font-size: 30px;
  }
}
```

- 显示器：1280 x 800
- ipad：1024 x 768
- Android：800 x 480
- iPhone：640 x 960
